Installation
Setting up Instafeed is pretty straight-forward - there are 3 main steps.  3단계 설치 과정 필요함

1. Create a Facebook app linked to Instagram, and add yourself as a test user. See Managing User Tokens.  페이스북 앱을 만들고 인스타그램을 연결
2. Create an access token and provide it to an Instagram Token service                                    접근코드 만들고 인스타 토큰 서비스를 제공받아야함
3. Add the instafeed.js script to your web page and provide some simple options. See Basic Usage          instafeed script 작성
