attr, prop 차이점
-attr : 속성이 무엇인지 1차원적인 값만 가져옴. 변수로 설정되어 있을경우 변수명만 가져옴
-prop : 본연의 속성까지 가져옴! 변수로 설정되어있더라도 본연의 속성까지 가져올수 있음

ready와 load의 차이점

1)웹브라우저 시작

2)웹문서 읽기

3)DOM생성 
  $(document).ready(function(){
    alert('준비가 되었습니다.')
  });

4)이미지를 포함한 요소들이 로드되기 시작함

5) 페이지 로딩완료
  $(document).load(function(){
    alert('내용이 모두 로드되었습니다.')
  })
